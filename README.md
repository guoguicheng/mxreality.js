### English Description

The SDK is available in three editions:

- [**Basic Edition (Free)**](https://github.com/guoguicheng/mxreality.js): Supports fundamental features including:
  - HLS streaming
  - MP4 video playback
  - Panoramic image viewing
  - Basic VR functionality

- [**Plus Edition (Free)**](https://github.com/guoguicheng/mxreality.js.plus): Includes Basic features plus:
  - Enhanced VR capabilities
  - Technical support package
  - Higher resolution support (up to 4K)
  - Extended codec support

- [**Pro Edition (Paid)**](//www.covideo.cn): Full enterprise solution with:
  - Ultra-low latency streaming
  - FLV streaming support
  - H.264/H.265 hardware acceleration
  - WebRTC integration
  - Stereoscopic 3D playback
  - Advanced customization options
